# Project: dance
## Purpose
A web-based video management and playback tool specifically for dancers. Features include CRUD operations for videos, advanced playback controls (mirroring, jumping by counts, section looping), and BPM-based rhythm training.

## Tech Stack (Web)
- React 19 (TypeScript)
- React Router DOM
- Vite
- Dexie.js (IndexedDB)
- Lucide React (Icons)
- Essentia.js (Audio/BPM analysis)
- Vanilla CSS

## Commands
- Dev: `npm run dev`
- Build: `npm run build`
- Lint: `npm run lint`
- Preview: `npm run preview`
